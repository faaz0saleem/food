Hungter DocumentationOverviewHungter is an AI-powered educational platform designed to democratize high-quality, personalized tutoring. By aggregating multiple LLM architectures (GPT, Claude, Gemini, Grok) into a single, gamified interface, Hungter provides students (K-12 to University) with a study experience tailored to their specific cognitive style.Core Value PropositionUnified Intelligence: Seamless access to multiple AI models in one chat interface.Gamified Retention: XP-based leveling, streaks, and progress tracking to build long-term learning habits.Adaptive Learning: Custom curriculum planning and AI-generated quizzes based on user progress.Tech StackFrontend: HTML5, CSS3, JavaScript (Core logic handling heavy UI state).Backend: Python (Django framework).Database: [Recommended: PostgreSQL] for robust handling of user data, history, and streaks.AI Integration: Centralized API Middleware (Proxy) to route requests between external AI providers (OpenAI, Anthropic, Google, xAI).Monetization Tiers (Business Logic)TierPriceFeaturesFree$020 msgs/day, 3 subjects, no saved progress.Student$5/moUnlimited messages, all subjects, progress tracking, access to all 4 engines.Family$12/moUp to 3 accounts, parent dashboard, detailed progress reports.School$200/mo50 student seats, teacher dashboard, custom curriculum management.Architectural RoadmapPhase 1: MVP Setup[ ] Initialize Django Project + Frontend repository.[ ] Configure User Authentication (Standard Django Auth + OAuth options).[ ] Implement the AI Routing Proxy: A backend middleware that receives the user's prompt and routes it to the selected AI engine (GPT/Claude/Gemini/Grok).[ ] Create local storage synchronization (transitioning localStorage to Database persistence).Phase 2: Engagement & Gamification[ ] Implement the XP/Leveling system logic (Server-side calculation to prevent cheating).[ ] Build the "Quiz Generator" engine (Prompt engineering pipeline for quiz creation).[ ] Develop the Parent/Teacher Dashboard (Reporting and analytics views).Phase 3: Advanced Intelligence[ ] Implement RAG (Retrieval-Augmented Generation) to give the AI long-term memory of the student's learning history.[ ] Connect to external LMS APIs (Google Classroom, etc.) for the School tier.Development EnvironmentClone the repository:
